WhatYouSeeIsWhatYouLearn (WYSIWYL) |version| documentation
==========================================================

What You See Is What You Learn is a powerful browser-based app that utilizes
autoML to give you a drag and drop interface to build powerful machine learning
pipelines, train and predict on new and existing data, and show the world your
models validity with publication ready visualizations!
